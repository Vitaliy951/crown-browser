# AI-Native Edge Browser (Прототип MVP)

Отечественный суверенный веб-браузер нового поколения со встроенной автономной системой искусственного интеллекта на базе локальных больших языковых моделей (LLM). 

Проект разработан в рамках импортозамещения в сфере ИТ и обеспечивает полную конфиденциальность данных пользователей при работе с генеративным ИИ за счет архитектуры граничных вычислений (**Edge Computing**).

## 📌 Ключевая концепция и ценность

Современные ИИ-ассистенты (ChatGPT, Copilot) отправляют персональные и корпоративные данные пользователей на зарубежные облачные серверы, что создает критические риски информационной безопасности. 

**AI-Native Edge Browser** решает эту проблему. Программа разворачивает ИИ-движок **локально на устройстве пользователя**. Весь инференс (вычисление ответов) происходит без использования интернета, что гарантирует:
1. **100% конфиденциальность:** данные не покидают периметр персонального компьютера.
2. **Экономическую устойчивость:** затраты на серверные мощности для ИИ равны 0 рублей, так как используются ресурсы процессора (CPU) и видеокарты (GPU) клиента.
3. **Умную монетизацию:** проект зарабатывает на нативной контекстной рекламе, которая интегрируется в ответы ИИ через внутренний промпт-инжиниринг без использования следящих куки-файлов (cookies).

## 🛠 Технологический стек

* **Оболочка и UI:** Node.js, Electron (Chromium Core / WebView2 API).
* **ИИ-Ядро:** Ollama Core API (интегрировано скрытым процессом).
* **Базовые модели:** Оптимизированные локальные модели (`llama3`, `phi3`, `gemma2`), кастомизируемые под требования пользователя.
* **Целевая ОС:** Microsoft Windows 10 / 11 (в дорожной карте — адаптация под отечественные ОС на базе Linux: Astra Linux, РЕД ОС).

## 🚀 Быстрый запуск (Инструкция по развертыванию MVP)

### Требования к окружению:
1. Установленная среда выполнения [Node.js](https://nodejs.org) (версии 18+).
2. Установленная локальная система [Ollama для Windows](https://ollama.com).
3. Предварительно загруженная базовая модель. Откройте терминал Windows (PowerShell) и выполните:
   ```bash
   ollama run llama3
   ```
   *(Убедившись, что модель скачана и отвечает, закройте консоль. Браузер сам запустит процесс в фоне).*

### Порядок установки приложения:
1. Клонируйте репозиторий с проектом:
   ```bash
   git clone https://github.com
   cd ai-native-browser
   ```
2. Установите необходимые зависимости:
   ```bash
   npm install
   ```
3. Запустите браузер в режиме разработчика:
   ```bash
   npm start
   ```

## 💼 Механизм монетизации (Рекламная модель)

В проекте реализован алгоритм **контекстного промпт-инжиниринга**. Главный процесс Node.js анализирует входящий запрос пользователя к ИИ. Если запрос связан с поиском ИТ-услуг, программного обеспечения или обучения, система дополняет системные инструкции для LLM.

В результате ИИ генерирует полезный ответ, в конец которого нативно и аккуратно встраивается рекомендация отечественного партнера или реферальная ссылка, что исключает необходимость демонстрации раздражающих баннеров.
